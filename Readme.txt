----使用方法----

後端:(路由為http://localhost:3005/)
1. 將team3.sql匯入MySQL資料庫
2. 將dev.env.sample改名為dev.env，並填入相關的資料庫參數
3. 執行npm i 安裝相關套件
4. 執行npm run dev


前端:(路由為http://localhost:3000/)
1. 執行npm i 安裝相關套件
2. 執行npm run dev
3. 前往http://localhost:3000 即可進到首頁

圖案包需至https://drive.google.com/drive/folders/1Iiv_2shRV0lsXpR8wGTyWkOdz5i85d5q?usp=drive_link下載
